Quiz Supercell

Projet: petit quiz interactif inspiré des jeux de la compagnie Supercell (ex: Clash Royale, Brawl Stars…). L’API (Flask) servira les questions; l’interface viendra plus tard.

Version de Python
- Python 3.13.x

Démarrage rapide (Windows / PowerShell)
1) Se placer dans le dossier du projet:
   cd "quiz-api"
2) Créer et activer l’environnement virtuel:
   py -m venv venv
   .\venv\Scripts\Activate.ps1
3) Installer les dépendances:
   pip install -r requirements.txt
   # ou depuis la racine du dépôt:
   pip install -r "quiz-api/requirements.txt"
4) Lancer le serveur de dev:
   python app.py

Par défaut, l’API écoute sur: http://localhost:5001/
